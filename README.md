## 🧠 Conceptos que se practican

- Manipulación de strings y caracteres (`ord`, `chr`, slicing)
- Estructuras de datos: listas, stacks, deques
- Algoritmos de ordenación con criterios múltiples
- Matrices y operaciones sobre listas 2D
- Manejo de casos borde (`None`, listas vacías, strings vacíos)

Cada carpeta contiene:
```

cd Exam03
```
